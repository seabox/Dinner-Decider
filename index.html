Setup required

The app isn't configured yet. Open app.js and set SUPABASE_URL and SUPABASE_ANON_KEY to your Supabase project values. Then follow these steps:

  1. Create a free project at supabase.com.
  2. Run the SQL in supabase-schema.sql in the Supabase SQL Editor.
  3. Enable the Azure OAuth provider in Authentication → Providers and add your Azure App Registration client ID & secret (see README for full Azure setup).
  4. Add your GitHub Pages URL (e.g. https://<user>.github.io/Dinner-Decider/) to Authentication → URL Configuration → Redirect URLs.
  5. Copy Project URL and anon public key from Project Settings → API into app.js.
  6. Commit and push — GitHub Pages will serve the updated site.
